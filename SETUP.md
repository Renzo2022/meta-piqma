# MetaPiqma - Plantilla Base

Plataforma integrada para revisiones sistemáticas y meta-análisis.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Estructura

- **src/App.jsx**: Componente principal con contexto, sidebar, header y módulos stub
- **src/index.css**: Estilos globales con tema Monokai Dimmed
- **tailwind.config.js**: Configuración de colores personalizados
- **vite.config.js**: Configuración de Vite

## Características Implementadas

✓ Tema Monokai Dimmed completo
✓ Sidebar con navegación a 8 módulos
✓ Context API con useReducer para estado global
✓ Animaciones con Framer Motion
✓ Iconos con Lucide React
✓ Scrollbar personalizado
✓ Responsive design
✓ Stub de apiClient para compatibilidad R/Python
